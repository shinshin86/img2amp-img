# img2amp-img

img to amp-img

```html
<img src="http://example.com/sample.png" alt="sample image" />
```

↓

```html
<amp-img
  alt="sample image"
  src="http://example.com/sample.png"
  width="900"
  height="675"
  layout="responsive"
>
</amp-img>
```
